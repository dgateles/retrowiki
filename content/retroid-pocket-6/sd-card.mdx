O Retroid Pocket 6 já vem com 128 GB de armazenamento interno — suficiente para começar. Mas com um catálogo de PS2, GameCube e Switch, esse espaço vai embora rápido. Um cartão microSD resolve a equação.

## Slot microSD

O RP6 tem **um slot microSD** acessível na lateral do console. Suporta cartões de até 2 TB (limite do padrão SDXC), mas na prática cartões de 256 GB a 512 GB atendem a maioria dos usuários.

<Alert type="warning" title="Cartões genéricos">
  Cartões sem marca vendem capacidade falsa com frequência — você acha que tem 256 GB mas corrompem dados após os primeiros gigabytes reais. Para ROMs de PS2 e GameCube (que chegam a 4-9 GB cada), use sempre marcas confiáveis.
</Alert>

## Formatação

O Android 13 do RP6 lê cartões formatados em **exFAT** sem problemas. Use exFAT para qualquer cartão acima de 32 GB.

### Windows

1. Abra o **Explorador de Arquivos** → clique com botão direito no cartão → **Formatar**
2. Sistema de Arquivos: **exFAT**
3. Tamanho da unidade de alocação: **128 KB** (boa escolha para arquivos grandes de ROMs)
4. Clique em **Iniciar**

### macOS

1. Abra o **Utilitário de Disco**
2. Selecione o cartão → **Apagar**
3. Formato: **ExFAT**
4. Clique em **Apagar**

### Linux

```bash
# Instale exfatprogs se necessário
sudo apt install exfatprogs

# Formate (substitua sdX pelo dispositivo correto — verifique com lsblk)
sudo mkfs.exfat -n "RP6-ROMS" /dev/sdX1
```

## Quanto espaço você precisa?

Referências de tamanho médio por sistema:

| Sistema | Tamanho médio por jogo | 100 jogos |
|---------|----------------------|-----------|
| NES / SNES / GBA | 1–4 MB | < 1 GB |
| PlayStation 1 | 300–700 MB | ~50 GB |
| Nintendo DS | 100–500 MB | ~30 GB |
| PSP | 500 MB–2 GB | ~100 GB |
| PlayStation 2 | 2–9 GB | ~400 GB |
| GameCube | 1–4 GB | ~200 GB |
| Nintendo Switch | 4–16 GB | ~700 GB |

Para uma coleção mista realista de PS2, GameCube e portáteis, **512 GB** costuma ser o ponto ideal. Se o foco for Switch também, considere 1 TB.

## Usando microSD como armazenamento adicionado

O Android permite usar o microSD como **armazenamento portátil** (padrão) ou **armazenamento interno** (formatação especial que mescla com a memória interna). Para ROMs e emuladores, o modo **portátil** é mais prático — você pode mover o cartão entre dispositivos facilmente.

Para configurar: **Configurações → Armazenamento → cartão SD → Usar como armazenamento portátil**.

## Estrutura de pastas sugerida

Organize as ROMs diretamente no cartão para que frontends como ES-DE e Daijisho encontrem automaticamente:

```
/sdcard/
  ROMs/
    NES/
    SNES/
    GBA/
    PlayStation/        ← PS1
    PlayStation2/
    GameCube/
    Wii/
    3DS/
    NDS/
    PSP/
    Switch/
  BIOS/
  Saves/
```
