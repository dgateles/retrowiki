## Layout físico

O RP6 tem dois layouts disponíveis — você escolhe no momento da compra:

- **Layout Switch-style**: sticks analógicos acima dos botões de face (esquerdo no lugar do D-pad)
- **Layout Xbox-style**: D-pad em cima à esquerda, sticks analógicos abaixo

Ambos incluem o mesmo conjunto completo de botões. A posição não pode ser alterada depois — é hardware físico, não configuração de software.

### Botões disponíveis

| Posição | Botões |
|---------|--------|
| Face | A, B, X, Y |
| Ombro superior | L1, R1 |
| Ombro inferior (gatilhos analógicos) | L2, R2 |
| Sistema | Home, Back, Volume +/- |
| Traseiros | M1, M2 (programáveis) |
| Analógicos | L3 (pressionar esquerdo), R3 (pressionar direito) |
| Extras | Touchscreen 5.5", giroscópio |

Os gatilhos L2/R2 são **analógicos** — respondem à pressão gradual, útil para jogos de corrida e similares.

Os sticks usam **Hall Effect** — tecnologia magnética sem contato físico, o que elimina o desgaste responsável pelo stick drift.

## Botões traseiros M1 e M2

O RP6 inclui dois botões traseiros programáveis (M1 e M2). Para mapeá-los:

1. Acesse **Configurações → Botões adicionais** (ou via app da Retroid)
2. Selecione M1 ou M2
3. Atribua qualquer botão do controle ou ação do sistema

Usos comuns: mapear M1 para L3 (pressionar stick esquerdo) em jogos onde isso é incômodo, ou M2 para captura de tela.

## Mapeamento no Android (jogos Android nativos)

Para jogos Android que não reconhecem controles por padrão, use o **Mantis Gamepad Pro** ou o **Panda Gamepad Pro** (disponíveis na Play Store). Eles criam um overlay de mapeamento que emula toque na tela.

Para jogos que reconhecem controles nativamente (como os da Netflix Games, emuladores, etc.), o RP6 aparece como gamepad padrão Android — sem configuração adicional.

## Hotkeys do RetroArch

O RetroArch usa o botão **Menu** (ou um botão configurado como hotkey) combinado com outros para executar ações rápidas.

### Configurar a hotkey

Em **Settings → Input → Hotkeys**, defina o botão "Enable Hotkey". O mais comum é usar **L3** ou **SELECT**.

### Atalhos durante o jogo

| Combinação | Função |
|-----------|--------|
| Hotkey + X | Abrir menu RetroArch |
| Hotkey + R1 | Salvar Estado (slot atual) |
| Hotkey + L1 | Carregar Estado (slot atual) |
| Hotkey + R2 | Avanço Rápido |
| Hotkey + L2 | Rebobinar (se habilitado) |
| Hotkey + B | Captura de tela |
| Hotkey + Y | Mostrar / ocultar FPS |
| Hotkey + A + B | Resetar jogo |
| Hotkey + START | Sair do jogo |
| Hotkey + D-pad cima/baixo | Mudar slot de save state |

### Atalhos do menu RetroArch

| Menu | Descrição |
|------|-----------|
| `Quick Menu → Cheats` | Códigos de trapaça |
| `Quick Menu → Shaders` | Filtros visuais (CRT, Scanlines) |
| `Settings → Input → Port 1` | Remapear controles por jogo |
| `Settings → Video → Scaling` | Proporção e integer scale |
| `Settings → Frame Throttle` | Velocidade de emulação |

## Hotkeys do Dolphin (GameCube / Wii)

O Dolphin para Android tem mapeamento próprio. Acesse **Settings → GameCube Input** ou **Wii Input** para configurar.

Hotkey padrão para abrir o menu durante o jogo: segure o botão **Back** por 2 segundos.

## Hotkeys do PPSSPP (PSP)

No PPSSPP, acesse **Settings → Controls → Control mapping** para configurar. Para sair do jogo, o padrão é tocar na tela para exibir o menu e selecionar "Back to menu".

## Giroscópio

O RP6 tem giroscópio integrado, útil para:

- **Jogos Switch com controle de movimento** (no Eden): configurar nas opções de mapeamento do emulador
- **Pokémon GO e similares** em modo nativo Android
- **Alguns jogos de Wii** no Dolphin (como mira com wiimote)

Para ativar no RetroArch: **Settings → Input → Sensors → Allow rotation input**.

## Touchscreen como controle

A tela de 5.5" funciona como touchscreen e tem usos práticos em emulação:

- **Nintendo DS no melonDS**: tela inferior do DS mapeada para a tela do RP6
- **Nintendo 3DS no Lime3DS**: mesma funcionalidade
- **Jogos Android com interface touch** que não suportam controles

<Alert type="info" title="Dica para DS e 3DS">
  No melonDS e Lime3DS, você pode tocar diretamente na tela do RP6 para simular a caneta stylus do DS — funciona muito bem para jogos como The World Ends With You e Professor Layton.
</Alert>
