## Compatibilidade por Sistema

| Sistema        | Desempenho | Observações                    |
| -------------- | ---------- | ------------------------------ |
| Game Boy / GBC | ✅ 100%    | Perfeito                       |
| GBA            | ✅ 100%    | Perfeito                       |
| SNES           | ✅ 100%    | Perfeito                       |
| Mega Drive     | ✅ 100%    | Perfeito                       |
| PS1            | ✅ 95%     | Maioria dos jogos              |
| N64            | ⚠️ 70%     | Mario 64 OK, GoldenEye lento   |
| Dreamcast      | ⚠️ 60%     | Jogos 3D pesados não rodam bem |
| PSP            | ⚠️ 50%     | Depende muito do jogo          |

## Dicas de Otimização

1. Use frameskip para jogos pesados
2. Reduza a resolução interna em emuladores 3D
3. Desative filtros de vídeo desnecessários
4. Use overclock se seu firmware suportar
