  Use cartões de marcas confiáveis como SanDisk, Samsung ou Kingston. Evite

## Cartões Recomendados

| Marca    | Modelo        | Capacidade | Status                 |
| -------- | ------------- | ---------- | ---------------------- |
| SanDisk  | Ultra         | 128GB      | ✅ Recomendado         |
| SanDisk  | Extreme       | 256GB      | ✅ Excelente           |
| Samsung  | EVO Select    | 128GB      | ✅ Recomendado         |
| Kingston | Canvas Select | 64GB       | ✅ Bom custo-benefício |
| Genérico | -             | Qualquer   | ❌ Não recomendado     |

## Formatação

Para formatar o cartão SD corretamente:

1. **Windows**: Use o [Rufus](https://rufus.ie/) ou [Balena Etcher](https://www.balena.io/etcher/)
2. **macOS**: Use o Balena Etcher
3. **Linux**: Use o comando `dd` ou Balena Etcher

<Alert type="warning" title="Atenção">
  A formatação apaga todos os dados do cartão. Faça backup antes de prosseguir.
</Alert>
