## Especificações Técnicas

| Especificação | Valor                            |
| ------------- | -------------------------------- |
| CPU           | Rockchip RK3326 Quad-core 1.5GHz |
| GPU           | Mali-G31 MP2                     |
| RAM           | 1GB DDR3L                        |
| Tela          | 3.5" IPS 640×480                 |
| Bateria       | 3200mAh (~6-8 horas)             |
| Armazenamento | microSD (até 512GB)              |
| Sistema       | Linux (ArkOS)                    |
