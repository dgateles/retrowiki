## Jogos Não Aparecem no Frontend

**Problema:** ROMs colocadas no cartão SD não aparecem no M Space, Daijisho ou ES-DE.

**Soluções:**
1. Confirme que as ROMs estão na pasta correta para o sistema (ex: `/ROMs/snes/` para SNES)
2. Verifique a extensão do arquivo — cada sistema aceita formatos específicos (`.sfc`, `.smc` para SNES; `.gba` para GBA; etc.)
3. Force um rescan: no M Space, vá em **Configurações → Atualizar Biblioteca**; no Daijisho, **Settings → Sync**; no ES-DE, pressione **Start → Game Collection Settings → Rescan**
4. Se o cartão SD for novo, verifique se está formatado em exFAT e montado corretamente (Configurações Android → Armazenamento)

## Áudio com Problema / Sem Saída P2

**Problema:** Fone de ouvido P2 (3.5mm) não funciona ou áudio sai distorcido.

Há relatos na comunidade de que em alguns units o P2 não funciona no firmware padrão. Passos para diagnosticar:

1. Teste com outro fone para descartar problema no acessório
2. Vá em **Configurações → Som → Saída de Áudio** e confirme que não está forçando saída para alto-falante
3. Tente reiniciar o console com o fone já conectado
4. Se o problema persistir no firmware de fábrica, considere instalar o **GammaOS Next** — a comunidade relatou melhorias de áudio nesse CFW

**Para alto-falantes com chiado ou volume baixo:**
- Ajuste o volume via botões físicos até ~80% (acima disso pode distorcer nos alto-falantes inferiores)
- Em RetroArch: **Settings → Audio → Volume Level** — mantenha em 100% e controle pelo volume do Android

<Alert type="info" title="Alto-falantes na base">
  Os alto-falantes ficam na parte inferior do console. Segurar o Air X horizontalmente pode bloquear parcialmente o som. Incline levemente o console ou use fones para sessões longas.
</Alert>

## Wi-Fi Instável ou Lento

**Problema:** Wi-Fi cai, conecta mas com velocidade baixa, ou não encontra redes 5 GHz.

1. O Air X tem Wi-Fi 5 (802.11ac) — confirme que seu roteador tem banda 5 GHz disponível
2. **Configurações → Wi-Fi → (nome da rede) → Esquecer** e reconecte
3. Se a conexão cair repetidamente: **Configurações → Bateria → Otimização de Bateria** → procure o processo de Wi-Fi e desative a otimização para ele
4. Para streaming de jogos (Xbox Cloud, GeForce NOW), a largura de banda do Wi-Fi 5 é suficiente, mas coloque o console próximo ao roteador para melhor estabilidade

## Controles Não Funcionam em Algum App

**Problema:** Botões físicos não respondem em determinado app ou jogo Android.

1. Verifique se o app suporta gamepad — muitos jogos mobile foram feitos somente para touch
2. Use o **Mantis Gamepad Pro** ou **GameKeyboard** para mapear botões a toques na tela
3. No RetroArch, confirme que o **Input Driver** está como `android` em **Settings → Drivers → Input**
4. Se um jogo específico do RetroArch não responde, vá em **Quick Menu → Controls → Port 1** e verifique se os botões estão mapeados

## Desempenho Baixo em Jogos

**Problema:** Jogo com slowdown, travadas ou quedas de frame.

**Passos gerais:**
1. Feche outros apps em segundo plano (botão de recentes no Android, deslize para fechar)
2. Ative o **Modo de Desempenho** em Configurações → Bateria
3. No RetroArch: desative shaders pesados se estiver usando (Quick Menu → Shaders → Remove)
4. Para GameCube (Dolphin): ative shaders pré-compilados e aguarde a compilação na primeira execução

**Expectativas realistas por sistema:**

| Sistema | Expectativa |
| ------- | ----------- |
| Até PSP / Dreamcast / N64 | 60fps estável na maioria dos títulos |
| Saturn | Maioria dos jogos roda bem; alguns títulos pesados têm drops |
| 3DS | Boa compatibilidade; jogos 3D intensos podem ter quedas |
| GameCube | Varia muito por jogo — títulos simples rodam, mas AAA como Mario Kart DD têm slowdown |
| PS2 | 15–30fps em títulos exigentes; jogos 2D rodam melhor |

## Console Trava / Não Responde

**Problema:** Tela congelada, botões não funcionam.

1. Pressione e segure o botão Power por **10 segundos** para forçar reinício
2. Se não responder ao botão Power, o Air X tem um botão de reset pequeno (pinhole) na parte inferior/lateral — use um objeto pontiagudo para pressionar
3. Após o reinício, verifique se o problema é específico a um app — se sim, limpe o cache desse app: **Configurações → Apps → (nome do app) → Armazenamento → Limpar Cache**

## Problemas ao Instalar GammaOS Next

**Problema:** Falha na instalação, console não inicializa após o flash.

<Alert type="warning" title="ROCKNIX e GammaOS não coexistem">
  Se você já instalou o ROCKNIX antes, ele afeta a capacidade de reiniciar normalmente. Antes de instalar o GammaOS Next, é necessário desconectar e reconectar a bateria internamente. Siga o guia oficial em retrohandhelds.gg.
</Alert>

1. Use cartões microSD de marcas confiáveis para a instalação — cartões genéricos causam falhas
2. Certifique-se de estar usando a imagem correta para o Air X (não confunda com versões para outros handhelds)
3. Em caso de brick, consulte o [canal Discord do GammaOS](https://github.com/TheGammaSqueeze/GammaOSNext) ou o grupo da comunidade

## Bluetooth com Latência Alta

**Problema:** Fone ou controle Bluetooth com atraso perceptível no áudio ou entrada.

1. Para fones Bluetooth em jogos, latência é inerente ao protocolo — prefira fones com aptX Low Latency
2. Para controles Bluetooth externos (8BitDo, PS4, Xbox), o delay é normalmente imperceptível em jogos de emulação
3. Mantenha o dispositivo Bluetooth próximo ao console para menor latência
4. Em RetroArch com controle Bluetooth: **Settings → Input → Input Latency Frames** → tente aumentar para 1-2 frames se sentir delay
