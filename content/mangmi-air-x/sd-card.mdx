O Air X tem 64 GB de armazenamento interno eMMC — suficiente para o sistema e emuladores, mas apertado para uma biblioteca de jogos. Um microSD é praticamente obrigatório se você pretende emular mais de um punhado de sistemas.

## Capacidade Recomendada

| Biblioteca | Capacidade Sugerida |
| ---------- | ------------------- |
| Sistemas 8/16-bit + GBA | 64 GB |
| + PS1, N64, DS, Dreamcast | 128 GB |
| + PSP, Saturn, GameCube parcial | 256 GB |
| Biblioteca completa | 512 GB |

Para a maioria dos usuários, **128 GB ou 256 GB** é o ponto ideal.

## Cartões Recomendados

| Marca | Modelo | Capacidade | Avaliação |
| ----- | ------ | ---------- | --------- |
| Samsung | Pro Plus | 128 GB / 256 GB | Excelente velocidade e confiabilidade |
| Lexar | Play / Professional | 128 GB / 256 GB | Boa velocidade para o preço |
| SanDisk | Extreme | 128 GB / 256 GB | Clássico confiável |

| Genérico / sem marca | — | Qualquer | Não recomendado |

<Alert type="warning" title="Cuidado com cartões falsos">
  Cartões genéricos e imitações de marcas conhecidas no AliExpress são comuns. Compre de vendedores com histórico positivo ou diretamente de lojas como Samsung ou SanDisk. Use o app H2testw (Windows) ou F3 (Linux/macOS) para verificar a capacidade real após a compra.
</Alert>

## Formatação

O Android reconhece microSD formatado em **exFAT** ou **FAT32**. Para cartões acima de 32 GB, use **exFAT**.

### Windows
1. Insira o cartão em um leitor de SD
2. Abra o Explorador de Arquivos → clique com botão direito no SD → **Formatar**
3. Selecione **exFAT** como sistema de arquivos
4. Marque **Formatação Rápida** e confirme

### macOS
1. Abra o **Utilitário de Disco**
2. Selecione o cartão → **Apagar**
3. Formato: **ExFAT**, Esquema: **Mestre de Inicialização** → Apagar

### Linux
```bash
sudo mkfs.exfat -n "MANGMI_SD" /dev/sdX1
```
(substitua `/dev/sdX1` pelo dispositivo correto — verifique com `lsblk`)

## Estrutura de Pastas

Após formatar, crie a estrutura de pastas para as ROMs. O padrão abaixo é compatível com ES-DE, Daijisho e o launcher nativo M Space:

```
/ROMs/
  arcade/
  dreamcast/
  gba/
  gb/
  gbc/
  gamecube/
  genesis/
  mastersystem/
  n64/
  nds/
  nes/
  psp/
  ps1/         (ou psx/)
  ps2/
  saturn/
  snes/
  3ds/
```

### Pasta de BIOS

Crie também uma pasta para os arquivos de BIOS do RetroArch:

```
/RetroArch/
  system/
    dc/          ← BIOS do Dreamcast aqui
    scph1001.bin ← BIOS do PS1
    bios7.bin    ← BIOS do Nintendo DS
    bios9.bin
    firmware.bin
```

## Inserindo o Cartão

O slot microSD fica na lateral do console. O cartão é encaixado com a face de contatos voltada para baixo. Após inserir:

1. O Android perguntará como usar o cartão — selecione **Armazenamento portátil** (não formate como armazenamento interno, isso reduz a velocidade de acesso)
2. Abra o gerenciador de arquivos e confirme que o SD aparece como dispositivo externo
3. Crie a estrutura de pastas e copie suas ROMs

<Alert type="info" title="Armazenamento portátil vs. interno">
  Evite a opção "Armazenamento interno" do Android ao inserir o SD. Ela usa criptografia e torna o cartão inutilizável em outros dispositivos, além de não oferecer ganho real de desempenho para ROMs.
</Alert>
