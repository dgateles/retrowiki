O botão modificador (hotkey) varia conforme o firmware: no **muOS** é o botão **MENU** (logo Anbernic); no **KNULLI**, o padrão é **SELECT** — mas pode ser reconfigurado em RetroArch → Settings → Input → Hotkeys.

## Durante o Jogo — muOS

| Combinação                                    | Função                         |
| --------------------------------------------- | ------------------------------ |
| <kbd>MENU</kbd> + <kbd>X</kbd>                | Abrir menu RetroArch           |
| <kbd>MENU</kbd> + <kbd>START</kbd>            | Sair do jogo (voltar ao menu)  |
| <kbd>MENU</kbd> + <kbd>R1</kbd>               | Avanço Rápido (Fast Forward)   |
| <kbd>MENU</kbd> + <kbd>R2</kbd>               | Salvar Estado (Save State)     |
| <kbd>MENU</kbd> + <kbd>L2</kbd>               | Carregar Estado (Load State)   |
| <kbd>MENU</kbd> + <kbd>Y</kbd>                | Mostrar/Ocultar FPS            |

## Durante o Jogo — KNULLI

Configuração padrão recomendada nos guias da comunidade:

| Combinação                                    | Função                              |
| --------------------------------------------- | ----------------------------------- |
| <kbd>SELECT</kbd> + <kbd>X</kbd>              | Abrir menu RetroArch                |
| <kbd>SELECT</kbd> + <kbd>START</kbd>          | Sair do jogo (voltar ao menu)       |
| <kbd>SELECT</kbd> + <kbd>R2</kbd>             | Avanço Rápido (Fast Forward)        |
| <kbd>SELECT</kbd> + <kbd>R1</kbd>             | Salvar Estado (Save State)          |
| <kbd>SELECT</kbd> + <kbd>L1</kbd>             | Carregar Estado (Load State)        |
| <kbd>SELECT</kbd> + <kbd>L2</kbd>             | Retroceder (Rewind — se habilitado) |

## Dentro do Menu RetroArch

Estes caminhos funcionam após abrir o RetroArch com <kbd>MENU</kbd>+<kbd>X</kbd> ou <kbd>SELECT</kbd>+<kbd>X</kbd>:

| Menu                                 | Função                              |
| ------------------------------------ | ----------------------------------- |
| `Quick Menu → State Slot`            | Alterar slot de save state          |
| `Quick Menu → Cheats`                | Códigos de trapaça                  |
| `Quick Menu → Overrides`             | Salvar configuração por jogo        |
| `Quick Menu → Shaders`               | Filtros visuais (CRT, Scanlines)    |
| `Settings → Video → Scaling`         | Proporção de tela (4:3, stretch)    |
| `Settings → Input → Port 1 Controls` | Remapear botões do controle         |
| `Settings → Audio → Latency`         | Buffer de áudio (padrão: 64ms)      |
| `Main Menu → Information → System`   | Informações de RAM e sistema        |

## Controles de Sistema

| Ação                             | Combinação / Botão              |
| -------------------------------- | ------------------------------- |
| Modo suspensão                   | <kbd>POWER</kbd> (toque curto)  |
| Desligar / Menu de energia       | <kbd>POWER</kbd> (toque longo)  |
| Ajuste de volume                 | Botões de volume laterais       |
| Ajuste de brilho (muOS)          | <kbd>MENU</kbd> + Botões volume |

## Frontend muOS

| Combinação                            | Função                                  |
| ------------------------------------- | --------------------------------------- |
| <kbd>A</kbd>                          | Selecionar / Confirmar                  |
| <kbd>B</kbd>                          | Voltar / Cancelar                       |
| <kbd>X</kbd>                          | Opções do item selecionado              |
| <kbd>Y</kbd>                          | Marcar favorito                         |
| <kbd>L1</kbd> / <kbd>R1</kbd>         | Navegar entre seções / sistemas         |
| <kbd>START</kbd>                      | Abrir menu de configurações             |
| <kbd>SELECT</kbd>                     | Alternar filtros de jogos               |

## Frontend KNULLI (EmulationStation)

| Combinação                      | Função                                 |
| ------------------------------- | -------------------------------------- |
| <kbd>START</kbd>                 | Abrir menu principal                   |
| <kbd>SELECT</kbd>                | Abrir menu de opções                   |
| <kbd>Y</kbd>                     | Seleção aleatória de jogo              |
| <kbd>X</kbd>                     | Opções do jogo selecionado             |
| <kbd>L1</kbd> / <kbd>R1</kbd>   | Alternar entre sistemas                |
| <kbd>L2</kbd> / <kbd>R2</kbd>   | Página acima/abaixo na lista de jogos  |
