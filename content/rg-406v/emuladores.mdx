A tela IPS 4:3 de **960x720** do RG-406V é um dos seus maiores trunfos para emulação: a proporção nativa elimina barras laterais em todos os sistemas retrô clássicos, e a resolução é múltiplo exato de 320x240 (3x), 256x224 (SNES, 4x) e 240x160 (GBA, 4x). Na prática isso significa pixels quadrados perfeitos sem interpolação.

## RetroArch

O RetroArch é a base recomendada para a maioria dos sistemas. No Android, baixe pela **Play Store** ou diretamente em retroarch.com.

### Configuração Inicial

1. Abra o RetroArch → **Online Updater → Update Core Info Files**
2. Baixe os cores pelo **Core Downloader**
3. Configure o diretório de ROMs em **Configurações → Diretório → Diretório de ROMs**
4. Coloque as BIOS em `RetroArch/system/` (armazenamento interno)

### Proporção de Tela no RetroArch

Para aproveitar a tela 4:3 ao máximo:

- **Configurações → Vídeo → Escala → Proporção de Aspecto**: selecione **4:3** ou **Core Provided**
- Para sistemas com pixel art: ative **Proporção Inteira** quando disponível
- Para shaders CRT: **Quick Menu → Shaders → Carregar Preset** → procure por `crt-royale` ou `crt-guest-dr-venom`

<Alert type="info" title="Tela 4:3 Nativa">
  A resolução 960x720 é exatamente **3x** o padrão NTSC (320x240) e **4x** o GBA (240x160). Ativar proporção inteira nesses sistemas resulta em pixels perfeitamente quadrados sem nenhuma interpolação.
</Alert>

## Emuladores Standalone por Sistema

### PlayStation 2 — NetherSX2

O NetherSX2 (fork do AetherSX2) é o emulador PS2 para Android mais estável atualmente. O RG-406V roda PS2 com boa consistência — a maioria dos jogos 2D e menos exigentes atingem 60 fps.

Configuração recomendada:
- **Backend de Renderização**: Vulkan
- **Renderer**: Hardware
- **Upscaling**: 2x (equilíbrio entre qualidade e desempenho)
- **Enable Cheats**: desativado por padrão

Jogos mais pesados (God of War, Shadow of the Colossus) podem ter quedas de frame — o score de **76** no JSON indica que a emulação é **boa**, não perfeita.

### GameCube e Wii — Dolphin

O Dolphin no RG-406V alcança score **85** — a maioria dos jogos roda bem com as configurações corretas:

- **Backend gráfico**: Vulkan (melhor desempenho)
- **Frequência da CPU**: nativa a 100%, ou overclock leve para jogos lentos
- **Antialias**: 0x (desative para ganho de performance)
- **Proporção**: 4:3 nativa para jogos antigos, 16:9 para jogos que suportam

### Nintendo 3DS — Lime3DS / Citra

Score **80** — a maioria dos jogos funciona, mas alguns títulos pesados como Xenoblade Chronicles 3D e Pokémon X/Y a 60fps podem exigir ajustes.

- Use **Lime3DS** (fork do Citra com melhorias para Android)
- Ative o JIT compiler
- Configure a resolução interna em **2x** para melhor visual sem perda de performance

### PSP — PPSSPP

Score **100** — o PSP roda perfeitamente. Configure:

- **Backend**: Vulkan
- **Resolução Interna**: 3x ou 4x (a tela 960x720 aproveita bem)
- **Frameskip**: 0 (desnecessário com o hardware do T820)

### Nintendo DS — melonDS ou DraStic

Score **100** — DS é um dos sistemas mais bem emulados. O melonDS no RetroArch oferece boa compatibilidade; para emulação standalone o DraStic (pago) tem a melhor compatibilidade.

A tela 4:3 do RG-406V é especialmente boa para DS — você pode configurar o layout dual-screen ou empilhado verticalmente.

### Saturn — Yabause / Kronos

Score **70** — emulação **playable** mas inconsistente. O Kronos (fork do Yabause) oferece melhor compatibilidade no RetroArch. Jogos 2D como Radiant Silvergun funcionam bem; jogos 3D pesados têm problemas.

### Nintendo Switch — Sudachi / Yuzu

Score **42** — emulação experimental. Jogos simples podem rodar, mas a maioria dos títulos Switch é exigente demais para o T820. Não é o ponto forte do RG-406V.

## Frontends Android

Um frontend organiza suas ROMs em uma interface dedicada, sem precisar navegar pelo launcher Android.

### Daijishō

O frontend mais popular para handhelds Android de emulação. Features:
- Scraping de artwork automático
- Suporte a todos os emuladores principais
- Temas personalizáveis
- Disponível no GitHub (sideload APK)

Configuração rápida:
1. Instale o APK do Daijishō
2. Adicione sua pasta de ROMs
3. Configure os emuladores para cada sistema
4. Execute o scraper para baixar artes

### Pegasus Frontend

Alternativa mais configurável, baseada em arquivos de texto para definir coleções. Ideal para quem quer controle total sobre a organização.

### Launchbox / BigBox (Android)

Versão Android do Launchbox com interface BigBox. Paga, mas com excelente scraping integrado.

<Alert type="info" title="RetroAchievements no Android">
  O RetroArch no Android suporta **RetroAchievements** normalmente. Crie uma conta em retroachievements.org e configure em **Configurações → RetroAchievements** dentro do RetroArch. O Wi-Fi 5 do RG-406V garante sincronização rápida.
</Alert>

## Configuração de Vídeo — Dicas para a Tela do RG-406V

A tela IPS de 300 PPI do RG-406V tem boa saturação e ângulo de visão. Para melhor experiência:

- **Brilho**: ajuste via botões de volume (Android) ou atalho do sistema
- **Shaders CRT**: funcionam bem na resolução 960x720 — experimente `crt-lottes` no RetroArch para uma aparência de tubo sem custo alto de performance
- **Integer scaling**: ative quando possível para evitar borramento
- **Filtro bilinear**: desative para pixel art — a resolução da tela é alta o suficiente para pixels nítidos
