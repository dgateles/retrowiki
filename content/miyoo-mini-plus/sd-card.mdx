## Capacidade e Formatação

O Miyoo Mini Plus suporta cartões microSD de até **512GB**, mas exige formatação
em **FAT32** obrigatoriamente — o OnionOS não inicializa em exFAT ou NTFS.
O Windows nativo não formata FAT32 em cartões acima de 32GB; use o SD Card
Formatter ou o Rufus. Para jogos retrô até PS1, **128GB ou 256GB** cobrem
facilmente qualquer coleção.

Use no mínimo Classe 10 / U1. Cartões lentos causam travamentos no boot e
lentidão nos menus. Certificação A1 melhora a resposta ao navegar pelo OnionOS.

## Formatação do Cartão SD

### Windows — SD Card Formatter (recomendado)

<Steps>
  <Step title="Baixe o SD Card Formatter">
    Acesse [sdcard.org/downloads/formatter](https://www.sdcard.org/downloads/formatter/) e instale. É a ferramenta oficial da SD Association e funciona para qualquer capacidade.
  </Step>
  <Step title="Selecione o cartão correto">
    Abra o SD Card Formatter e confirme a letra do drive antes de prosseguir. Um engano aqui apaga o disco errado.
  </Step>
  <Step title="Selecione Overwrite Format e formate">
    Escolha **Overwrite Format** (obrigatório para cartões já usados) e clique em **Format**. Aguarde a conclusão.
  </Step>
</Steps>

### Windows — Rufus (cartões acima de 32GB)

1. Abra o [Rufus](https://rufus.ie/) e selecione o cartão SD
2. Em **Boot selection**, escolha `Non bootable`
3. Mude **File system** para `FAT32`
4. Clique em **Start** e confirme

### macOS

1. Abra o **Disk Utility**, selecione o cartão na barra lateral
2. Clique em **Erase** e escolha o formato **MS-DOS (FAT)**
3. Confirme e aguarde

Se os jogos aparecerem duplicados no OnionOS, o macOS criou arquivos ocultos
`._*` ao copiar. Corrija com:

```bash
dot_clean /Volumes/NOME_DO_CARTAO
```

### Linux

```bash
# Verifique o dispositivo correto com lsblk antes de executar
sudo mkfs.vfat -F 32 /dev/sdX1
```

## Estrutura de Pastas do OnionOS

Após instalar o OnionOS, a raiz do cartão terá esta estrutura:

```
/
├── Roms/           ← ROMs organizadas por sistema (GBA, PS, SNES…)
├── Saves/          ← Save states e saves in-game
├── BIOS/           ← Arquivos de BIOS (PS1, NDS, Neo Geo)
├── Emu/            ← Configurações dos emuladores
├── App/            ← Aplicativos extras
└── miyoo/          ← Arquivos do sistema — não modifique
```

O OnionOS usa nomes padronizados para as subpastas: `GBA` (Game Boy Advance),
`PS` (PlayStation 1), `SNES` (Super Nintendo), `NDS` (Nintendo DS), `MD`
(Mega Drive). A lista completa aparece dentro de `Roms/` após a instalação.
