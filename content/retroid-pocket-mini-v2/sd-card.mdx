O RPMini V2 já vem com 128 GB de armazenamento interno UFS 3.1 — velocidade de leitura/escrita bem acima do microSD comum. Para a maioria dos usuários, isso é suficiente para uma biblioteca completa dos sistemas clássicos até PS2. O slot microSD existe para expandir quando a biblioteca crescer.

## Você Precisa de um Cartão SD?

Depende. 128 GB comporta:

| Acervo estimado | Tamanho aproximado |
|----------------|-------------------|
| NES + SNES completos | ~5 GB |
| GBA (No-Intro completo) | ~10 GB |
| PS1 (seleção curada, ~200 jogos) | ~20–30 GB |
| PSP (seleção, ~100 jogos) | ~15–30 GB |
| GameCube (10–15 jogos) | ~15–20 GB |
| PS2 (10–15 jogos) | ~15–30 GB |

Se você curar a biblioteca e não quiser todos os jogos existentes de cada sistema, o interno é suficiente. Cartão SD vale a pena para quem quer ROMs No-Intro completas de vários sistemas ou uma coleção grande de GameCube/Wii/PS2.

## Cartões Recomendados

Use cartões de marcas conhecidas. Cartões genéricos podem ter velocidade de escrita inconsistente e corromper dados.

| Marca | Modelo | Capacidade | Avaliação |
|-------|--------|------------|-----------|
| Samsung | Pro Plus | 128 GB / 256 GB | Excelente velocidade, confiável |
| Samsung | Evo Plus | 128 GB / 256 GB | Boa relação custo-benefício |
| SanDisk | Extreme | 128 GB / 256 GB / 512 GB | Sólido, amplamente testado |
| Lexar | Play | 256 GB / 512 GB | Focado em gaming, boa velocidade |
| Kingston | Canvas Go Plus | 128 GB | Bom custo-benefício |
| Genérico | Qualquer | Qualquer | Não recomendado |

Para jogos de GameCube e Wii (arquivos grandes de 1–4 GB cada), prefira um cartão com velocidade de leitura acima de 90 MB/s. Cartões A2 (Application Performance Class) têm melhor desempenho para acesso aleatório, útil para emuladores.

## Formatação

O Android 13 do RPMini V2 trabalha bem com **exFAT** para cartões acima de 32 GB.

### No Windows

1. Abra o **Explorador de Arquivos**, clique com o botão direito no cartão e selecione **Formatar**.
2. Sistema de arquivos: **exFAT**
3. Tamanho da unidade de alocação: **128 KB** (para arquivos grandes como ISOs de GameCube/PS2, aumentar para 512 KB ou 1 MB reduz fragmentação)
4. Clique em **Iniciar**.

Alternativa: use o **SD Card Formatter** (ferramenta oficial da SD Association, gratuita) para formatação mais limpa.

### No macOS

1. Abra o **Utilitário de Disco**.
2. Selecione o cartão e clique em **Apagar**.
3. Formato: **ExFAT**, Esquema: **GUID** ou **Master Boot Record**.
4. Confirme.

<Alert type="warning" title="Atenção">
  Formatar apaga todos os dados. Se o cartão já tem ROMs ou saves, faça backup antes de prosseguir.
</Alert>

## Montando o Cartão no Android

Ao inserir um cartão exFAT novo no RPMini V2, o Android pergunta como você quer usá-lo:

- **Armazenamento portátil** — acesso normal como cartão removível. Esta é a opção recomendada para ROMs.
- **Armazenamento interno** — formata o cartão como extensão do armazenamento interno (perde compatibilidade com outros dispositivos). Não recomendado para uso com emuladores.

Selecione **Armazenamento portátil**.

## Estrutura de Pastas no SD Card

Se você separa ROMs no cartão e outras no armazenamento interno, mantenha a mesma estrutura de pastas nos dois locais para facilitar a configuração dos emuladores:

```
/sdcard/  (cartão microSD)
└── Roms/
    ├── Gamecube/
    ├── Wii/
    ├── PS2/
    ├── PSP/
    └── ...
```

No RetroArch, você pode adicionar múltiplos diretórios de conteúdo por sistema. No Daijisho, aponte cada plataforma para a pasta correspondente (suporta múltiplos caminhos).

## Velocidade e Desempenho

Para a maioria dos sistemas (até PS1 e N64), qualquer cartão de marca confiável funciona. Para GameCube, Wii e PS2, jogos grandes podem se beneficiar de cartões mais rápidos:

- ISOs de GameCube têm até 1.35 GB — carregam rápido mesmo em cartões medianos
- ISOs de PS2 chegam a 4+ GB — carregamento inicial pode demorar 5–10 segundos em cartões lentos

O armazenamento interno UFS 3.1 é consideravelmente mais rápido que qualquer microSD. Se tiver jogos pesados que usa com frequência, mantenha eles no interno.
