O Snapdragon 865 do RPMini V2 é sólido o suficiente para GameCube, Wii e PS2 com folga. A tela AMOLED de 3.92" e proporção 31:27 é quase quadrada — isso é uma vantagem enorme para sistemas 4:3 como SNES, PS1 e N64, que ocupam praticamente toda a área de display sem barras laterais visíveis.

## RetroArch

O RetroArch é a base para a maioria dos sistemas clássicos. Instale pela Play Store ou baixe a versão mais recente em [retroarch.com](https://www.retroarch.com).

### Cores recomendados por sistema

| Sistema | Core recomendado | Observação |
|---------|-----------------|------------|
| NES | Mesen | Melhor compatibilidade e precisão |
| SNES | Snes9x | Mais leve que bsnes; excelente |
| Game Boy / GBC | Gambatte | Paletas precisas |
| Game Boy Advance | mGBA | Padrão ouro para GBA |
| Mega Drive / Genesis | Genesis Plus GX | Compatibilidade quase total |
| PlayStation 1 | Beetle PSX HW | Hardware rendering; requer BIOS |
| Nintendo 64 | Mupen64Plus-Next | Ativar Vulkan para melhor desempenho |
| Dreamcast | Flycast | Configure com Vulkan |
| Nintendo DS | melonDS | Suporte nativo a tela dupla |
| Saturn | Beetle Saturn | Playable; alguns jogos com bugs |
| PSP | PPSSPP (standalone) | Melhor usar o app direto |

### Configurações de vídeo para a tela AMOLED

A resolução 1240x1080 e proporção 31:27 pedem atenção ao scaling:

- **Integer Scale**: ative para sistemas pixel-art. O SNES em 256x224 escala inteiramente para 1024x896, centralizando na tela sem distorção.
- **Shaders**: o AMOLED tem contraste nativo altíssimo, então shaders de CRT ficam espetaculares. Experimente `crt-royale` ou `zfast-crt` (mais leve).
- **HDR/cores vívidas**: o painel AMOLED do RPMini V2 exibe pretos verdadeiros — jogos escuros como Castlevania SOTN ficam notavelmente melhores do que em IPS.

<Alert type="info" title="Proporção 31:27">
  A tela do RPMini V2 é quase quadrada. Jogos 4:3 (a grande maioria dos clássicos) ficam perfeitos sem qualquer stretch. Jogos 16:9 como PSP e alguns Wii terão pequenas barras horizontais — é o comportamento esperado para manter a proporção correta.
</Alert>

## Emuladores Standalone

Para sistemas mais exigentes, os emuladores standalone têm vantagem de desempenho em relação aos cores RetroArch.

### Dolphin (GameCube / Wii)

O Dolphin Emulator está disponível na Play Store. Com o Snapdragon 865 e Adreno 650:

- **GameCube**: a grande maioria dos jogos roda em full speed (60 fps). Score 100/100 no benchmarks de compatibilidade.
- **Wii**: também excelente. Jogos 3D como Mario Galaxy e Zelda: Skyward Sword rodam bem com configurações médias.

Configurações recomendadas:
- Backend gráfico: **Vulkan**
- Resolução interna: **2x** (bem acima do nativo, ainda confortável para o hardware)
- Sync on Skip Idle: **ativado**
- MMU Speed Hack: **ativado** (melhora compatibilidade)

### PPSSPP (PSP)

Instale pela Play Store. O PSP tem score perfeito no RPMini V2 — o Adreno 650 lida com os jogos PSP com facilidade.

- Backend: **Vulkan**
- Rendering Resolution: **3x PSP** (1334x750 — boa relação qualidade/desempenho na tela de 3.92")
- Texture Filtering: **Anisotropic 8x**
- Post-processing shader: experimente `fxaa` ou `natural colors`

### AetherSX2 / NetherSX2 (PS2)

O PS2 tem score 88/100 — a maioria dos jogos roda bem, alguns títulos mais pesados podem precisar de ajustes.

- O **NetherSX2** (fork atualizado do AetherSX2) tem melhor compatibilidade. Instale via APK.
- Renderer: **Vulkan**
- Upscaling: **2x** é um bom equilíbrio
- Hacks de compatibilidade: ative por jogo quando necessário (tela de patches)

### Citra (Nintendo 3DS)

Score perfeito no RPMini V2. Use o **Citra** ou o **Lime3DS** (fork mantido pela comunidade).

- Rendering Resolution: **3x Native (768p)** — a tela AMOLED valoriza bastante o upscaling
- A tela dupla do 3DS fica bem com o layout horizontal do console

### DraStic (Nintendo DS)

O melhor emulador de NDS para Android. Disponível na Play Store (pago) ou use o core melonDS no RetroArch.

A proporção quasi-quadrada da tela do RPMini V2 é ideal para o layout dual-screen do DS — ambas as telas ficam confortáveis.

## Saturn

Score 70/100 — jogável, mas com ressalvas. O Beetle Saturn (RetroArch) é a opção mais compatível, mas o Saturn é notoriamente difícil de emular. Jogos 2D como Street Fighter Alpha e Radiant Silvergun rodam bem; jogos 3D têm mais variação.

## Nintendo Switch, Wii U e PS3

Scores baixos (40, 20 e 10 respectivamente) — não são alvos práticos para o RPMini V2. O Snapdragon 865, embora potente para emulação clássica, fica abaixo do que esses sistemas demandam de forma consistente.

## Frontends

Um frontend organiza sua biblioteca com arte, metadados e navegação unificada. As três opções mais usadas na comunidade Retroid:

### Daijisho

O favorito da comunidade para Android. Interface limpa, suporte a scraping automático de arte, temas customizáveis e integração transparente com qualquer emulador instalado.

Instalação:
1. Baixe na Play Store.
2. Na primeira execução, adicione plataformas e aponte para a pasta de ROMs de cada sistema.
3. Execute o scraper integrado para baixar capas, screenshots e metadados.

Vantagens no RPMini V2: a tela AMOLED faz as capas dos jogos parecerem pôsteres — a experiência visual é marcante.

### ES-DE (EmulationStation Desktop Edition)

Versão moderna do EmulationStation, disponível para Android. Mais familiar para quem vem de handhelds Linux (RG35XX, R36S etc.).

- Suporte a temas visuais elaborados
- Scrapers integrados (ScreenScraper, IGDB)
- Navegação limpa por sistema

### Pegasus Frontend

Mais minimalista e altamente configurável via arquivos de texto. Melhor para usuários que querem controle total sobre a organização da biblioteca e preferem configurar manualmente.

Todos os três frontends funcionam bem com o RPMini V2. A escolha é de preferência pessoal — se você vier de um handheld Linux, o ES-DE é a transição mais natural; se for novo no ecossistema, o Daijisho tem a menor curva de aprendizado.
