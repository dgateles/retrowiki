## Console aquece muito durante emulação pesada

O Snapdragon 865 é um chip poderoso e gera calor durante GameCube, Wii e PS2. O RPMini V2 tem cooling ativo, mas em sessões longas o aquecimento é normal.

**O que fazer:**
- Ative o modo de **desempenho sustentável** no emulador (Dolphin: Configurações → Geral → Limite de CPU)
- Reduza a resolução interna no emulador (de 3x para 2x no Dolphin, por exemplo)
- Faça pausas de 5–10 minutos a cada hora em jogos muito exigentes
- Verifique se a ventilação traseira não está bloqueada (não jogue em superfícies macias como cama)

Se o console reduzir o desempenho automaticamente (thermal throttling), o jogo vai gaguejar. Isso é proteção do hardware — não é defeito.

## Emulador trava ou fecha sozinho

**Causas mais comuns:**

1. **Falta de BIOS**: PS1, PS2, Dreamcast e Saturn precisam de arquivos BIOS. Verifique se estão na pasta correta (`/RetroArch/system/`).
2. **ROM corrompida**: teste outra ROM do mesmo jogo.
3. **Versão desatualizada do emulador**: atualize pela Play Store.
4. **Memória insuficiente**: feche outros apps antes de iniciar emuladores pesados. O RPMini V2 tem 6 GB de RAM, mas apps Android em segundo plano consomem parte disso.

Para verificar memória disponível: **Configurações → Sobre o dispositivo → Memória RAM**.

## Áudio com chiado, delay ou sem som

**Áudio com chiado ou artefatos:**
- No RetroArch: Configurações → Áudio → Latência de Áudio. Aumente para 128ms ou 256ms se houver crackling.
- Driver de áudio: tente trocar de `OpenSL ES` para `AudioTrack` em Configurações → Áudio → Driver.

**Delay no áudio (input lag):**
- Diminua a latência de áudio no RetroArch para 64ms ou menos
- Certifique-se de que o V-Sync está ativo no emulador
- Use fones via P2 ao invés de Bluetooth para latência menor

**Sem som via Bluetooth:**
- Bluetooth tem latência inerente (~100–200ms). Para jogos rítmicos, prefira fones com fio.
- Se o fone Bluetooth parece não funcionar: vá em Configurações → Bluetooth, desemparelhe e emparelhe novamente.

## Sticks com drift ou descalibrados

Os sticks Hall Effect do RPMini V2 não têm drift mecânico por desgaste, mas podem precisar de calibração:

1. Vá em **Configurações → Dispositivos Conectados** (ou busque "Calibração" no sistema)
2. Execute a calibração com os sticks na posição central e depois movendo até os extremos
3. No RetroArch, verifique a zona morta (dead zone) em Configurações → Entrada → Port 1 → Zonas Mortas do Analógico. Um valor entre 0.05 e 0.15 costuma ser ideal.

Se o drift persistir após calibração, pode ser firmware — verifique se há atualização disponível.

## Wi-Fi desconecta ou tem velocidade baixa

O RPMini V2 tem Wi-Fi 6. Se estiver com problemas:

- **Roteador antigo (Wi-Fi 4/5)**: compatível, mas certifique-se que o roteador não está em modo de economia de energia
- **Canal congestionado**: se possível, mude o canal do roteador para 5 GHz
- **IP fixo**: em redes com muitos dispositivos, configurar IP fixo no Android pode ajudar (Configurações → Wi-Fi → rede atual → IP Estático)
- **Scraping lento no Daijisho/ES-DE**: limite o scraping para não sobrecarregar o servidor — 3–5 requisições por segundo é o máximo recomendado

## Tela com burn-in ou manchas

A tela AMOLED tem pequena susceptibilidade a burn-in com imagens estáticas por longos períodos (ex: HUD sempre visível no mesmo lugar).

**Prevenção:**
- Ative o protetor de tela do Android quando o console estiver inativo
- Use o modo escuro do Android — pixels pretos no AMOLED ficam desligados, sem desgaste
- Evite deixar o menu parado por horas sem uso

Se aparecerem manchas ou pixels mortos durante o período de garantia, entre em contato com a Retroid via [goretroid.com](https://www.goretroid.com).

## Bateria não carrega ou carrega lento

<Alert type="warning" title="Use o cabo e carregador corretos">
  O RPMini V2 suporta carregamento rápido via USB-C. Use um carregador PD (Power Delivery) de pelo menos 18W para carregamento normal. Carregadores de 5V/1A (antigos) vão carregar, mas lentamente.
</Alert>

- **Carrega lento**: verifique se o carregador suporta PD. Um carregador de celular moderno com USB-C PD funciona bem.
- **Não carrega**: tente outro cabo USB-C. Cabos genéricos de baixa qualidade podem não suportar a corrente necessária.
- **Bateria drenando rápido**: reduza o brilho da tela AMOLED (é o maior consumidor), desative Wi-Fi quando não usar e feche apps em segundo plano.

## Jogos PS2 com gráficos glitchados

O PS2 tem score 88/100 — a maioria funciona bem, mas alguns jogos precisam de ajustes específicos:

- Em AetherSX2/NetherSX2, acesse **Patches** para o jogo específico e ative o patch de compatibilidade recomendado pela comunidade
- Troque o renderer de Vulkan para OpenGL (alguns jogos têm bugs específicos em um ou outro)
- Verifique a compatibilidade na lista oficial do AetherSX2/NetherSX2 no GitHub

## RetroAchievements não funciona

- Verifique se está logado: RetroArch → Configurações → RetroAchievements → Nome de Usuário / Senha
- O Wi-Fi precisa estar ativo durante o jogo
- Nem todos os jogos têm achievements — verifique no site [retroachievements.org](https://retroachievements.org)
- Se o login falhar, verifique se a conta está ativa no site
