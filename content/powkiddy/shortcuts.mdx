## Hotkeys do Sistema (ArkOS / JELOS / ROCKNIX)

Os três firmwares compartilham a mesma base EmulationStation, então as combinações abaixo funcionam em todos eles.

| Combinação                                       | Função                          |
| ------------------------------------------------ | ------------------------------- |
| <kbd>SELECT</kbd> + <kbd>X</kbd>                | Abrir menu RetroArch            |
| <kbd>START</kbd> + <kbd>SELECT</kbd> (2x)       | Sair do jogo                    |
| <kbd>POWER</kbd> (toque curto)                  | Modo suspensão                  |
| <kbd>POWER</kbd> (toque longo)                  | Menu desligar                   |

## Atalhos do EmulationStation (Menu)

| Combinação        | Função                       |
| ----------------- | ---------------------------- |
| <kbd>START</kbd> | Abrir menu principal/sistema |
| <kbd>SELECT</kbd>| Abrir menu de opções         |
| <kbd>A</kbd>     | Confirmar / iniciar jogo     |
| <kbd>B</kbd>     | Voltar                       |
| <kbd>Y</kbd>     | Favoritar item               |

## Hotkeys do RetroArch

<kbd>SELECT</kbd> funciona como tecla modificadora em qualquer core RetroArch.

| Combinação                                       | Função                        |
| ------------------------------------------------ | ----------------------------- |
| <kbd>SELECT</kbd> + <kbd>X</kbd>                | Abrir menu RetroArch          |
| <kbd>SELECT</kbd> + <kbd>R1</kbd>               | Salvar Estado (Save State)    |
| <kbd>SELECT</kbd> + <kbd>L1</kbd>               | Carregar Estado (Load State)  |
| <kbd>SELECT</kbd> + <kbd>R2</kbd>               | Avanço Rápido                 |
| <kbd>SELECT</kbd> + <kbd>L2</kbd>               | Retroceder (se habilitado)    |
| <kbd>SELECT</kbd> + <kbd>B</kbd>                | Captura de tela               |
| <kbd>SELECT</kbd> + <kbd>Y</kbd>                | Mostrar/Ocultar FPS           |
| <kbd>SELECT</kbd> + <kbd>START</kbd>            | Sair do jogo                  |

<Alert type="info" title="Múltiplos slots de save state">
  O RetroArch permite múltiplos slots de save state por jogo. Alterne entre eles
  em `Quick Menu → State Slot` antes de salvar ou carregar.
</Alert>

## Menus úteis do RetroArch

| Menu                               | Descrição                          |
| ---------------------------------- | ---------------------------------- |
| `Settings → Video → Shaders`       | Filtros visuais (CRT, Scanlines)   |
| `Settings → Video → Scaling`       | Proporção de tela (4:3, 1:1, etc.) |
| `Quick Menu → Cheats`              | Códigos de trapaça                 |
| `Settings → Input → Port 1`        | Mapeamento de controles            |
| `Settings → Audio → Latency`       | Latência de áudio                  |
| `Quick Menu → State Slot`          | Alterar slot de save state         |
| `Settings → Achievements`          | Habilitar RetroAchievements (Wi-Fi)|

<Alert type="warning" title="Emuladores standalone (ex.: PPSSPP)">
  PSP e alguns outros sistemas usam emuladores standalone com hotkeys próprias.
  Se <kbd>SELECT</kbd> + <kbd>START</kbd> não sair do jogo, consulte as
  configurações do emulador específico.
</Alert>
