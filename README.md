## Sobre

O Retro Wiki é uma documentação colaborativa mantida pela comunidade, focada em consoles de emulação portáteis como R36S, Miyoo Mini, RG35XX e outros dispositivos similares.

## Consoles Documentados

- **R36S** - Firmware, detecção de clones, atalhos, performance
- **Miyoo Mini** - Em breve
- **RG35XX** - Em breve
- **TrimUI** - Em breve
- **Powkiddy** - Em breve

## Contribuindo

A documentação é construída em MDX. Para contribuir:

1. Fork este repositório
2. Edite ou crie arquivos `.mdx` na pasta `content/`
3. Abra um Pull Request

### Estrutura de Arquivos

```
content/
├── docs/           # Página inicial
├── r36s/           # Documentação R36S
│   ├── index.mdx
│   ├── firmware.mdx
│   ├── shortcuts.mdx
│   └── ...
├── miyoo/          # Documentação Miyoo
├── rg35xx/         # Documentação RG35XX
└── ...
```

### Formato MDX

Os arquivos seguem o formato MDX com frontmatter YAML:

```mdx
---
title: Título da Página
description: Descrição que aparece no cabeçalho e SEO.
---

## Conteúdo aqui
```
